# Java: Criando a sua Primeira Aplicação

> O curso aborda os fundamentos da linguagem Java, compreendendo desde a configuração do ambiente de desenvolvimento e o processo de compilação na JVM até o domínio de estruturas de controle e tipos de dados.

> Este repositório foca na consolidação da sintaxe Java através do desenvolvimento de projetos, desafios, manipulação de dados e controle de fluxo.
